# **Dear Diary - Integrated Project**

**Dear Diary** is a psychological horror game website featuring an interactive diary, character lore exploration, and community hub. The project aims to create an immersive experience where users can explore game mysteries through clickable diary items.

## **Design Process**

The design focuses on a **horror/diary aesthetic** with torn paper effects, blood-red accents, and a dark, mysterious atmosphere. The target audience includes horror game fans who enjoy interactive storytelling and community engagement.

### **Current User Stories**
* As a player, I want to click on diary items with "?" marks to discover hidden clues
* As a user, I want to navigate between different pages (Home, World, Merch, Hub, Login)
* As a mobile user, I want a responsive design that works on different devices

### **Wireframes**
* Based on provided Figma designs showing interactive diary layout
* Mobile-first approach with sandwich navigation

## **Current Features**

### **Implemented Features**
* **Multi-page Structure** - Basic HTML structure for all required pages (Home, World, Merch, Hub, Login)
* **Navigation System** - Sandwich menu icon and login/signup button in navbar
* **Basic Styling** - Dark theme with horror elements implemented in CSS
* **File Organization** - Proper folder structure with CSS, HTML, and planned image folders

### **Features in Development**
* **Interactive Diary** - Clickable "?" items that will show popups with clues
* **Character Profiles** - World page with clickable character files
* **Community Forum** - Hub page with discussion cards and user posts
* **Authentication** - Login/Signup forms with validation
