# Dear Diary - Integrated Project

Dear Diary is a psychological horror game website featuring an interactive diary where users can click on mysterious items to uncover clues about the missing character Wei Yi.

## Design Process

The design focuses on creating a haunting, personal atmosphere similar to discovering someone's private diary. The target audience includes horror game enthusiasts who enjoy unraveling mysteries through interactive storytelling.

### User Stories
* As a mystery solver, I want to click on diary items marked with "?", so that I can uncover hidden clues
* As a lore enthusiast, I want to browse character files, so that I can understand their backstories
* As a mobile user, I want to navigate using the sandwich menu, so that I can access all pages on my phone
* As a new visitor, I want to sign up for the newsletter, so that I can receive exclusive updates

### Wireframes
* Based on provided Figma designs showing the interactive diary interface
* Mobile-first responsive design approach

## Features

### Existing Features
* **Multi-page Structure** - HTML files for Home, World, Merch, Community Hub, and Login pages
* **Responsive Navigation** - Sandwich menu and login button in navbar
* **Horror-themed Design** - Dark color scheme with blood-red accents and paper textures
* **Mobile Container** - Centered layout for immersive mobile experience

### Features Left to Implement
* **Interactive Diary** - Clickable "?" items that reveal clues through modal popups
* **Character Profiles** - Expandable character files with detailed backstories
* **Community Forum** - Discussion cards for fan theories
* **Authentication** - Login/Signup forms with validation
* **Merchandise Store** - Product listings and shopping functionality
* **JavaScript Interactivity** - All interactive functionality
